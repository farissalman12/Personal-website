# Faris Salman - Personal Portfolio

This is a personal portfolio website for **Faris Salman**, built with HTML, CSS (Bootstrap), and jQuery. It showcases skills in React Native, CyberSecurity, and Graphic Design.

## Project Structure

-   `index.html`: The main entry point for the website.
-   `css/`: Stylesheets (Bootstrap, custom styles).
-   `js/`: JavaScript files (jQuery, plugins, custom logic).
-   `img/`: Images for the portfolio, testimonials, and assets.
-   `fonts/`: Web fonts used in the template.

## How to Edit

1.  **Content**: Open `index.html` and modify the text within the relevant sections (Hero, About, Resume, etc.).
2.  **Styles**: Custom styles can be added to `css/custom.css`.
3.  **Scripts**: Custom logic (like navigation handling) allows in `js/custom.js`.

## Deployment

This is a static website. You can deploy it to:
-   **GitHub Pages**: Go to Settings > Pages > Source: `main` branch / root.
-   **Netlify/Vercel**: Drag and drop the folder or connect your GitHub repo.
